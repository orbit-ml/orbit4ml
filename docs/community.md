# Community

orbit4ml is an open project. Contributions, feedback, and discussion are welcome.

## Get Involved

- **GitHub** — [github.com/orbitml/orbit4ml](https://github.com/orbit-ml/orbit4ml) — open issues, submit PRs, or start a discussion.
- **Discussions** — Use [GitHub Discussions](https://github.com/orbit-ml/orbit4ml/discussions) for questions, ideas, and showcasing your work.

## Contributing

Contribution guidelines will be published with the first release. In the meantime, feel free to open an issue or discussion to share ideas or flag gaps in the ecosystem.

## Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) code of conduct. Be kind, be constructive.
