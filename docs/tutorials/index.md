# Tutorials

Step-by-step guides that walk you through orbit4ml's core concepts and workflows.

## Available Tutorials

- **[Getting Started](getting_started.md)** — Install orbit4ml, create a constellation, run your first orbital simulation, and train a model under space constraints.

## Planned Tutorials

- **Working with Space Datasets** — Load and transform Earth observation and telemetry data with `orbit4ml.data`.
- **Constraint-Aware Training** — Use power and thermal profiles to schedule training across eclipse cycles (v0.2).
- **Compressing Models for Downlink** — Reduce model size for bandwidth-constrained missions (v0.3).
- **Federated Learning Across Constellations** — Train models across satellites with dynamic topology (v1.0).
- **Benchmarking Your Method** — Use `orbit4ml.bench` to evaluate and compare approaches fairly (v0.2).
