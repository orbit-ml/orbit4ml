# Community

orbit4ml is an open project. Contributions, feedback, and discussion are welcome.

## Get Involved

- **GitHub** — [github.com/orbit-ml/orbit4ml](https://github.com/orbit-ml/orbit4ml) — open issues, submit PRs, or start a discussion.
- **Discussions** — Use [GitHub Discussions](https://github.com/orbit-ml/orbit4ml/discussions) for questions, ideas, and showcasing your work.

## Contributing

We welcome contributions of all kinds — bug reports, feature requests, documentation improvements, and code.

### Quick Start

```bash
# Clone the repo
git clone https://github.com/orbit-ml/orbit4ml.git
cd orbit4ml

# Install in development mode
pip install -e ".[dev]"

# Run the tests
pytest
```

### Guidelines

- **Tests required** — all new features and bug fixes should include tests
- **Docstrings** — use Google-style docstrings for all public APIs
- **Type hints** — all function signatures should be typed
- **One PR per feature** — keep pull requests focused and reviewable

## Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) code of conduct. Be kind, be constructive.
